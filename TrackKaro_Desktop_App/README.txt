# Track-Karo Desktop App

## 🚀 Quick Start Guide

### How to Run the App:
1. **Double-click `new_app.exe`** to start the application
2. The Track-Karo dashboard will open automatically
3. Use your organization credentials to log in

### First Time Setup:
- The app will connect to the Track-Karo backend automatically
- Make sure you have an internet connection
- If Windows shows a security warning, click "More info" → "Run anyway"

## 📋 System Requirements

**Your computer needs:**
- Windows 10 (version 1903) or Windows 11
- 4GB RAM (8GB recommended)
- Internet connection
- 64-bit Windows (most computers after 2010)

## 🔧 Installation

**No installation required!** Just:
1. Extract this folder to your computer
2. Double-click `new_app.exe`
3. Start using the app

**Optional:** Create a desktop shortcut:
- Right-click on `new_app.exe`
- Click "Send to" → "Desktop (create shortcut)"

## 🆘 Troubleshooting

**App won't start?**
- Make sure you have internet connection
- Try right-clicking `new_app.exe` → "Run as administrator"
- Restart your computer and try again

**Windows security warning?**
- Click "More info"
- Click "Run anyway"
- This is normal for new apps

**App is slow?**
- Close other programs to free up memory
- Check your internet connection speed
- Restart the app
